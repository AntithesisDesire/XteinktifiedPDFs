✦ What is this tool for?
⬩➤ PDF Xteink-ify breaks down high-resolution PDFs into a bunch of images that essentially "crops" them to smaller 480x800 pixels, preserving quality so that the dithering in the Xteink doesn't affect them. Afterwards, you can use this PDF and convert it to XTC format using another tool.

✦ How does it work?
⬩➤ 1: Choose a PDF file from your local storage.
⬩➤ 2: Render your magnification scale (higher = more zoom and more crop, lower = low zoom and less crop)
⬩➤ 3: Choose a format (PDF/Zip of PNG files).
⬩➤ 4: Set up a page range for your PDF (leave it blank if you want the tool to render the whole PDF).
